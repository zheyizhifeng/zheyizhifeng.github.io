修改主题以及配置

oh-my-zsh 的配置文件位于 ~/.zshrc，可以进行自定义的修改配置。

主题配置

ZSH_THEME 字段为主题字段，可选的主题参考 ~/.oh-my-zsh/themes/ 目录。

插件配置

本文推荐 3 个较为常用的 oh-my-zsh 插件

插件简介

  • zsh-autosuggestions

    输入命令时可提示自动补全

  • zsh-syntax-highlighting

常用的命令会高亮显示，命令错误显示红色

  • git

插件默认启用，支持各种 git 命令简写, 参考 ~/.oh-my-zsh/plugins/git/git.plugin.zsh
